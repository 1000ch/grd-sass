# grd-sass

Sass port of 1000ch/grd that is a CSS grid framework using Flexbox.

## Install

```bash
# via npm
$ npm install grd-sass

# via bower
$ bower install grd-sass
```

## Usage

You can import `grd.scss` and use `grid` and `cell` as `@mixin`.
